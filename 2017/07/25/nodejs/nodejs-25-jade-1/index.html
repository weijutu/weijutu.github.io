Jade 特點

  • Jade 用縮排來定義 HTML 的階層結構
  • Jade 比 HTML 寫起來更簡潔
  • Jade 在編譯後，會將 HTML 的補上結束標籤
  • Jade 不需要寫小於大於的符號
