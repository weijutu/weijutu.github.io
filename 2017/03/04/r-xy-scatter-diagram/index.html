R 語言 - 模擬身高體重並繪製 X-Y 散佈圖

2017-03-04

試做：使用 N(172, 5)模擬 100人身高、使用 N(58, 5)模擬 100 人，繪製 X-Y 散佈圖

XY 散佈圖是指：

XY 圖的基本形式是以一個資料數列為基礎，而這個資料數列包含名稱、X 軸值清單以及 Y 軸值清單。每個值配對 (x|y) 在座標系統中都會顯示為一點。

學習目標

透過 R 軟體畫出 XY 散佈圖。

語法

接著畫出身高體重的 XY 散佈圖

結果

線性迴歸分析

height 為「被解釋變數」
weight 為「解釋變數」

結果

迴歸分析 F 檢定 p-value = 0.7176
