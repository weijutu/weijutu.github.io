租用專屬自己的 Linode VPS 虛擬主機

發表於 2020-09-01 更新於 2022-09-04 分類於 工具

喜歡寫網站的我經常有不少應用程式開發的想法在腦中飄蕩，為了一步步的落實它們，那麼就需要有個理想中的系統環境預備驗證開發上的操作，關於主機如何規劃這件事，也讓我想了一些方案

組電腦 vs 租電腦的選擇

原本是希望在自己的 Mac 裡面裝一個 Linux 虛擬主機做開發，但 Mac 本身是 SSD 的儲存空間空間卻不大，但是如果還要做遷移到雲主機的話，變成還要進行遷移，為了在這開發的路線能夠減少重複性的工作，我需要的是一台可以隨時隨地有一個立刻就可以使用的機器，如果組一台電腦，這樣就還需要將電腦隨時開著，沒有使用時會覺得滿浪費電的，所以我稍微分析了組電腦跟租電腦可能會遇到的優缺點

組電腦的優缺點

  • 可以自己挑選想要的廠牌與規格、預估價格
  • 如果電腦不工作時會關機，那麼下次要同步資料時，需要開機後才做同步
  • 備份自己切磁區來做，或是將資料備份到另一顆硬碟之類
  • 故障找廠商，會有保固時間
  • 要設定固定 IP 給主機，一直要開機有點麻煩

租電腦的優缺點

  • 每個月付租用費
  • 只要有網路就可以做連線，可以隨需求選擇開機或關機甚至備份等
  • 故障了，就重置虛擬主機
  • 網路速度快慢問題
  • 有點門檻，要學 Linux 能力

經過分析對於自己的需求來說，基於開發的方便性，那麼決定使用租電腦的方式來實現。

為什麼選擇租用 Linode

其實有非常多家的選擇，比如 Digital Ocean、Godaddy 之類的，不過地點也滿接近的所以選擇 Linode ，基本上是一個兼具便宜又啟動快速的虛擬主機，以價格來看，最小的規格 1C1G 一個月只需要 5 美金，的確小型租用的話滿划算的。

註冊使用 Linode

首先一開始會進入到 Linode 官網 在右上角的「Sign Up」按鈕做一個建立帳號，有三個步驟要做，(1) 建立帳號、(2) 填寫付款資訊、(3) 完成註冊
