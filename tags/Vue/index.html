6 еҲҶйҗҳ й–ұи®Җж–Ү (еӨ§зҙ„ 914 еҖӢеӯ—)

еңЁ Mac дёҠ Apache йғЁзҪІ Vue з¶Із«ҷ

еҹәжң¬дёҠ Mac дҪңжҘӯзі»зөұиЈЎйқўе·Із¶“жңүе…§е»әзҡ„ ApacheпјҢжүҖд»Ҙжң¬зҜҮж–Үз« е°ұдёҚжҸҗеҰӮдҪ•е®үиЈқ Apache зҡ„дјәжңҚеҷЁгҖӮеңЁдҪҝз”Ё Apache зҡ„жҷӮеҖҷйңҖиҰҒжңү root зҡ„ж¬ҠйҷҗпјҢи«Ӣе…Ҳијёе…Ҙд»ҘдёӢзҡ„жҢҮд»ӨгҖӮ

ps. Apache дјәжңҚеҷЁзүҲжң¬зӮәпјҡApache/2.4.23 (Unix)

жҺҘи‘—еҶҚијёе…Ҙ Apache зҡ„е•ҹеӢ•жҢҮд»Ө

йҖҷжҷӮеҖҷжңғзңӢеҲ°жңҚеӢҷе·Із¶“ијүе…Ҙе•ҹеӢ•

з„¶еҫҢзңӢдёҖдёӢ http://localhost/ жңүжІ’жңүеҮәзҸҫ It works! еӯ—зңјпјҢе°ұд»ЈиЎЁ Apache жңүжӯЈеёёйҒӢиЎҢпјҢиҖҢйҖҷеҖӢз¶Ій Ғзӣ®йҢ„жҳҜж”ҫеңЁ /Library/WebServer/Documents

Apache еҸғж•ёиЁӯе®ҡ

дҝ®ж”№ httpd.conf

ијёе…Ҙе®Ңз•ўд№ӢеҫҢпјҢеңЁ /etc/apache2/httpd.conf жҠҠд»ҘдёӢзҡ„й …зӣ®зҡ„иЁ»и§Ј # з§»йҷӨ

дҝ®ж”№ httpd-userdir.conf

еңЁ httpd з·ЁијҜе®Ңз•ўд№ӢеҫҢпјҢеҶҚдҝ®ж”№ /etc/apache2/extra/httpd-userdir.conf жӘ”жЎҲпјҢе°Үд»ҘдёӢзҡ„й …зӣ®зҡ„иЁ»и§Ј # з§»йҷӨ

йҖҷжЁЈеӯҗпјҢйғҪз·ЁијҜе®Ңз•ўеҶҚе°Ү Apache йҮҚе•ҹгҖӮйҖҷжҷӮеҖҷйҖІе…Ҙ http://localhost/ е°ұжңғеҫ—еҲ°д»ҘдёӢз•«йқўгҖӮ

еҰӮжһңеҮәзҸҫ You don t have permission to access / on this server д№ҹе°ұд»ЈиЎЁиЁӘе•Ҹзҡ„дҪҚзҪ®жҳҜжІ’жңүж¬Ҡйҷҗзҡ„гҖӮеҸҜд»Ҙе°ҮиіҮж–ҷеӨҫиӘҝж•ҙдёҖдёӢж¬ҠйҷҗпјҢ(еҰӮжһңз¶Із«ҷж”ҫеңЁ /Projects/demo)

Vue з¶Із«ҷзҡ„дҝ®иЁӮ

дҝ®ж”№ config/index.js

жүҫеҲ° build е…§зҡ„ assetsPublicPath еҸғж•ёпјҢеҺҹжң¬й җиЁӯеҖјжҳҜ /пјҢиҖҢйҖҷеҖӢи·Ҝеҫ‘е°Қйқңж…Ӣзҡ„иіҮжәҗж–Үд»¶жҳҜзө•е°Қи·Ҝеҫ‘гҖӮ

ж”№жҲҗзӣёе°Қи·Ҝеҫ‘зҡ„еҜ«жі•пјҢе°Ү assetsPublicPath еҸғж•ёж”№зӮә ./

дҝ®ж”№ build/utils.js

жҺҘи‘—йҖІе…Ҙ build/utils.js ж”№ css дҪҚзҪ®зҡ„и·Ҝеҫ‘ publicPath

дҝ®иЁӮ index.pug

жүҫеҲ°з¶Із«ҷж №зӣ®йҢ„зҡ„ index.pug жӘ”жЎҲпјҢдҝ®ж”№ /static/favicon.ico зӮә ./static/favicon.ico

дҝ®иЁӮ vue-router

з”ұж–јжҲ‘еҖ‘еёҢжңӣз¶ІеқҖжҳҜ http://localhost/demo/ пјҢиҰҒеңЁ route й…ҚзҪ®иЈЎйқўеҠ дёҠ base еҸғж•ёгҖӮ

йғЁзҪІеҲ°дјәжңҚеҷЁ

дёҖеҲҮйғҪе°ұз·’д№ӢеҫҢпјҢйҖІе…ҘеҲ°ж №зӣ®йҢ„дёӢеҹ·иЎҢ

build еҮәдҫҶз¶Із«ҷжңғз”ўз”ҹдёҖеҖӢжү“еҢ…еҘҪзҡ„ dist зҡ„иіҮж–ҷеӨҫпјҢйҖҷжҷӮеңЁ Apache зҡ„з¶Із«ҷз©әй–“е…§пјҢеҒҡдёҖеҖӢ link жҢҮеҗ‘йҖҷеҖӢдҪҚзҪ®

ps. иЁҳеҫ—еҫҢйқўзҡ„ /demo дёҚиҰҒеӨҡдёҖеҖӢж–ңз·ҡ /demo/ дёҚз„¶жңғжүҫдёҚеҲ°гҖӮ

йҖҷжЁЈе°ұе®ҢжҲҗеӨ§йғЁеҲҶзҡ„иЁӯзҪ®дәҶгҖӮ

ж–°еўһ .htaccess жӘ”жЎҲ

дҪҶжҳҜпјҢеҸҲзӮәд»ҖйәјиЁӯзҪ®е®Ңз•ўд№ӢеҫҢпјҢз¶Із«ҷеҚ»еҸҲдёҖзүҮз©әзҷҪпјҹд№ҹжҳҜеӣ зӮә Vue зҡ„ history жЁЎејҸжҰӮеҝөпјҢдёҚжңғиҮӘеӢ•ең°зҷјиө· http и«ӢжұӮпјҢи§Јжұәж–№жі•е°ҮжүҖжңүзҡ„и«ӢжұӮйҖІиЎҢиҪүзҷјеҲ° http://localhost/demo/index.html

еҸҜд»ҘеңЁз¶Із«ҷзӣ®йҢ„еӨҫиЈЎйқўж–°еўһ .htaccess жӘ”жЎҲпјҢе…§е®№ијёе…Ҙд»ҘдёӢ

йҖҷжЁЈе°ұиғҪз”Ё http://localhost/demo/ дҫҶиЁӘе•Ҹз¶Із«ҷдәҶгҖӮ
